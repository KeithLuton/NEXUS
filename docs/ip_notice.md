## "FreewareLayer" terminology

The README refers to a **FreewareLayer** — this is a naming convention,
not a separate license. The FreewareLayer consists of:

- The orchestrator core (`core/`)
- The Python SDK / validation tools (`tools/`)
- Community-contributed hardware configs (`config/vendors/`)
- Documentation (`docs/`)

All of the above are Apache 2.0 licensed and free to use, modify, and
redistribute under that license.
